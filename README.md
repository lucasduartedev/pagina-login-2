# 🔐 Projeto - Página de Login (Simulação de Autenticação)

## 📋 Sobre o Projeto

Este projeto consiste em uma página de login desenvolvida com HTML5, CSS3, JavaScript e jQuery para simular a autenticação de usuários.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- jQuery

## ✨ Funcionalidades

- Interface moderna e responsiva
- Validação de campos obrigatórios
- Simulação de autenticação assíncrona
- Mensagens de sucesso e erro
- Limpeza automática da senha após falha
- Feedback visual durante a validação

## ▶️ Como Executar

1. Abra o arquivo `index.html` em um navegador.
2. Informe as credenciais de teste.
3. Clique em **Entrar**.

## 🔄 Fluxo da Aplicação

1. Validação dos campos.
2. Exibição de carregamento.
3. Verificação das credenciais.
4. Retorno de sucesso ou erro.
